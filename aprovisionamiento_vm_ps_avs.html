Instrucciones de Ejecución

      
Genera el script utilizando el formulario de la izquierda.

      
Copia el script generado o descárgalo como archivo .ps1.

      
Abre PowerShell como administrador.

      
Asegúrate de tener instalado el módulo Az.VMware:

      
Install-Module -Name Az.VMware -Force

      
Ejecuta el script con el siguiente comando (reemplaza  con la ubicación real del archivo):

      
.\\Create-AzureVMwareVMs.ps1
