Execution Instructions

  1. Generate the script using the form on the left.

  3. Open your preferred Terraform editor or IDE.
  4. Create a new Terraform configuration file and paste the script content.
  5. Run terraform init to initialize the Terraform working directory.
  6. Run terraform apply to provision the Azure resources.
