Instrucciones de Ejecución

  1. Genera el script utilizando el formulario a la izquierda.
  2. Copie el script generado o descárgalo como un archivo .tf.
  3. Abre tu editor o IDE de Terraform preferido.
  4. Crea un nuevo archivo de configuración de Terraform y pega el contenido del script.
  5. Ejecuta terraform init para inicializar el directorio de trabajo de Terraform.
  6. Ejecuta terraform apply para provisionar los recursos en Azure.
