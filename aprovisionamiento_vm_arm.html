Instrucciones de Ejecución

      
    
        
  1. Genera el script usando el formulario arriba.
    2. 
        
  2. Copie el script generado o descárguelo como un archivo .json.
    3.

    4.

    5. 
        
  5. En la CLI de Azure, utiliza el comando az deployment group create --resource-group  --template-file .json.
